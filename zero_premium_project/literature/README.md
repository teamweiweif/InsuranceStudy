# Literature

This folder stores the local Drake et al. 2025 article and supplementary files used to audit the descriptive replication.

Files:

- `drake_2025_jama_health_forum_article.pdf`: main article.
- `drake_2025_supplement_1.pdf`: supplement with eAppendix/eTable details, including missing-data handling and excluded counties.
- `drake_2025_supplement_2.pdf`: data-sharing supplement.

Reference:

Drake C, Nagy D, Avina S, Ludwinski D, Anderson DM. Coverage Retention and Plan Switching Following Switches From a Zero- to a Positive-Premium Plan. JAMA Health Forum. 2025;6(5):e251424. doi:10.1001/jamahealthforum.2025.1424.
